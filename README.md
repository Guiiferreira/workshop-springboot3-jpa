# Workshop Spring Boot 3 com JPA

Este repositório contém o código e os exemplos do workshop de integração entre Spring Boot 3 e JPA (Java Persistence API).

## Tecnologias Usadas
- Spring Boot 3
- JPA
- H2 Database
- PostgreSQL
- POSTMAN
- Maven
